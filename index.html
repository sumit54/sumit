Java Object Initilizer

Blueprint of Class

Object is chunk of memory bundled with code which manipulate memory in which state and behaviour (variables and methods) are available. Every Object require specific memory to process with its state and behaviour.

JVM provides memory to object where JVM runs initialization object code first and allocate default memory to object. Object default value is null to allocate particular memory, we need to initialize object by “new” keyword. Without object we cannot access class’s variables and methods.

Example

public class ClassObject {
data_type variable;
void method() {
//Do Anything
}
public static void main(String[] args) {
ClassObject object=new ClassObject();
object.method();
}
}

Object Class

Superclass for every class in JAVA

Object class is root class (Super class) for every class in JAVA.Every class directly (if class doesnot extends any class)or indirectly (if class extends any other class) derived from Object class.

java.lang package imported into Object class so we dont need to worry about it.Without this package one cannot complie any program.

Object Class has number of methods which are available to every classes in JAVA.
